--- layout: default title: Javascript 04 permalink: /js/04.html/ --- # Javascript 04 Professor: Rodrigo Chaves --- # Orientação a Objetos 2 --- # Classes --- `prototype` --- # Construtor ```js function Apple (type) { this.type = type; this.color = "red"; } ``` --- # Métodos ```js Apple.prototype.getInfo = function() { return this.color + ' ' + this.type + ' apple'; }; ``` --- # O que são expressões regulares O JavaScript tem uma ferramenta predefinida extremamente poderosa chamada expressão regular. Essa ferramenta permite a seleção de cadeias de caracteres de interesse em um texto. Expressões regulares podem, também, serem conceituadas como uma sequência de caracteres que definem um padrão de busca. --- As expressões regulares surgiram no ano de 1956 quando Stephen Cole Kleene descreveu linguagens regulares usando notação matemática. A teoria definida pelo matemático ganhou enorme importância na área teórica da ciência da computação, principalmente na teoria dos autômatos e na teoria das linguagens formais.
