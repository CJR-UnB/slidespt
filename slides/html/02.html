A cor dos elementos pode ser escrita usando o nome da cor. O problema é que dessa forma temos um número muito pequeno de possíveis cores. Para aumentar a diversidades de cores podemos utilizar o valor rgb daquela cor ou o valor hexadecimal. --- # Width Essa propriedade define o width de um elemento. Ela pode assumir diferentes valores como por exemplo: auto, length, initial, inherit. ``` img { width: 100px; } ``` ``` img { width: 10em; } ``` ``` img { width: 50%; } ``` ??? Quando usamos auto o browser calcula o width do elemento. Length corresponde a passar o tamanho do width, seja com pixels, ems, ou porcentagens. Initial utiliza o valor igual ao default daquele elemento. Inherit faz aquele elemento herdar o width do elemento pai. --- # Height Essa propriedade é similar ao width. ``` img { height: 100px; } ``` ``` img { height: 10em; } ``` ``` img { height: 50%; } ``` --- # Background ``` body { background-color: pink; } ``` ``` body { background-image: url("path.jpg"); } ``` + https://css-tricks.com/almanac/properties/b/background-image/ + [An good stackoverflow question](http://stackoverflow.com/questions/2643305/ centering-a-background-image-using-css) --- # Box model ![Box model]({{ base-url }}/assets/images/html/box_model.png) --- # margin ``` p { margin: 10px 20px 30px 40px; } ``` ``` p { margin-top: 10px; margin-right: 20px; margin-bottom: 30px; margin-left: 40px; } ``` --- # padding ``` p { padding: 10px 20px 30px 40px; } ``` ``` p { padding-top: 10px; padding-right: 20px; padding-bottom: 30px; padding-left: 40px; } ``` --- # border ``` p { border: 2px solid green; } ``` ``` p { border-width: 2px; border-style: solid; border-color: green; } ``` --- # Modificando o texto + Text Color + Define a cor de um elemento. ``` body { color: green; } ``` + Text Align + Um texto pode estar alinhado com o centro (center), com as bordas (right/left), ou pode estar definido para que suas linhas ocupem exatamente o mesmo tamanho (justified). ``` h3 { text-align: justify; } ``` --- + Text Decoration + Vários elementos HTML vem com estilos predefinidos. Dentre eles, um dos mais irritantes para desenvolvedores novatos é o text-decoration de elementos como por exemplo links. Aqueles underlines raramente são necessários e por isso é possível se livrar deles da seguinte forma: ``` a { text-decoration: none; } ``` Além de tirá-lo é possível substituí-lo por outros estilos que se adequam à situação. Seguem exemplos uteis: ``` h1 { text-decoration: overline; text-decoration: underline; text-decoration: line-through; } ``` --- # Exercício 3 ![Tarefa 3]({{ base-url }}/assets/images/html_exercises/tarefa3.png) [Imagens]({{base-url}}/download/html/exercises/tarefa3.zip) --- # Referências + https://developer.mozilla.org/en-US/docs/Web/HTML/Element + https://developer.mozilla.org/en-US/docs/Web/Guide/HTML/Introduction +
