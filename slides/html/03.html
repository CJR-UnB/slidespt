... ``` {% highlight css %} /*no arquivo box.css*/ .box { width: 100px; height: 100px; position: static; } ... } {% endhighlight %} --- Faz o elemento ter um comportamento normal, que é a de ser deixado em sua posição atual no fluxo da página. As propriedades top, right, bottom, left e z-index não se aplicam para elementos com posição static. --- {% highlight css %} /*no arquivo box.css*/ .box { ... position: relative; } ... } {% endhighlight %} --- `relative` se comporta igualmente ao static, a menos que se adicione propriedades extras no estilo do elemento. porém... {% highlight css %} /*no arquivo box.css*/ .box { ... position: relative; top: 10px; left: 10px; } ... } {% endhighlight %} Existe `bottom` e `right` também. --- Definindo as propriedades top, right, bottom, e left em um elemento position:relative o ajustamos em relação à posição atual dele. Outros conteúdos não vão se ajustar para se encaixarem em qualquer espaço deixado por esse elemento. --- {% highlight css %} /*no arquivo box.css*/ .box { position: fixed; bottom: 0; right: 0; width: 200px; background-color: white; } ... } {% endhighlight %} --- Um elemento fixo - fixed - é posicionado relativamente ao "viewport", isso significa que ele sempre ficará no mesmo lugar mesmo que haja rolagem na página. Assim como o relative, as propriedades top, right, bottom e left também são utilizadas. --- {% highlight css %} /*no arquivo box.css*/ .box { ... position: absolute; top: 10px; left: 10px; } ... } {% endhighlight %} --- Este valor se comporta como o fixed, porém tendo como referência a posição do elemento relativo mais próximo de onde está contido, ao invés do viewport. Se um elemento position:absolute não possuir elementos ancestrais posicionados relativamente, ele utilizará o body como referência. --- # float A propriedade float permite que um elemento flutue para uma determinada posição. Quando usamos `float: left;` o elemento flutua para esquerda da página. O oposto ocorre ao usarmos `float: right;`. --- {% highlight css %} /*no arquivo box.css*/ .box { ... float: right; } ... } {% endhighlight %} --- {% highlight css %} /*no arquivo box.css*/ .box { ... float: left; } ... } {% endhighlight %} --- {% highlight css %} /*no arquivo box.css*/ .clearfix { overflow: auto; } {% endhighlight %} --- # Referências + https://developer.mozilla.org/pt-BR/docs/Web/CSS/position
