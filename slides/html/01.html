O HTML é uma linguagem de marcação utilizada na construção de páginas Web. --- # HTML = Hipertext Markup Language --- # O que é uma linguagem de marcação Não adianta nada linkar e relacionar informações na web se essas informações não tiverem significado. Os browsers precisam de ajuda para diferenciar um parágrafo de um título, e é ai que vem a marcação. Ela inclui tags que possibilitam a diferenciação dos elementos da página. --- ## Hipertext É um texto ao qual se agregam outros conjuntos de informações. Ele facilita a navegação dos internautas. Na web, um texto pode ter diversas palavras que, ao serem clicadas, são remetidos para outra página onde se esclarece com mais precisão o assunto do link anterior. --- # Web Browsers --- # Conceitos básicos --- # Servidor Web Aprender sobre isso. --- # Protocolo O que são os protocolos de Web? + HTTP (Hypertext Transfer Protocol) O que é HTTP? + FTP (File Transfer Protocol) = Transferência de arquivos.
