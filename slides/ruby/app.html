--- layout: default title: Ruby App permalink: /ruby/app/ --- # Ruby App Tutor: Rafael Fernandes --- ## Aplicação Desenvolver uma aplicação básica que gerencie os livros de uma biblioteca. A aplicação deve ter uma classe `Livro` com os atributos: + nome + codigo (inteiro) + autor E uma classe `Biblioteca` com os atributos: + livros (hash) Os livros devem ser criados e armazenados na biblioteca no formato: `{ codigo: livro (objeto) }` --- As operações que devem ser feitas na biblioteca: + adicionar livros + remover livros + retornar todos os livros Faça um menu que forneça as seguintes opções para o usuário: + criar novo livro (e já o insere na biblioteca) + exibir biblioteca + exibir tamanho da biblioteca + excluir livro da biblioteca
